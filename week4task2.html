Alphabetical Order JavaScript Function

Here is the JavaScript function that returns a passed string with letters in alphabetical order:

<script>
    function alphabeticalOrder(str) {
        return str.split('').sort().join('');
    }

    let exampleString = 'webmaster';
    let sortedString = alphabeticalOrder(exampleString);
    console.log(sortedString); // Expected Output: 'abeemrstw'
</script>
